# Governance Committee Meeting Minutes

## Scenario

Documented the outcomes and action items from an IT Governance Committee meeting.

## Deliverable

- Formal meeting minutes
- Action tracking
- Governance documentation

## Skills Demonstrated

- Minute Taking
- Business Writing
- Action Tracking
- Stakeholder Coordination

## Framework Alignment

- ISO 27001
- COBIT

## Topics Covered

- Policy Review
- User Access Review
- Security Awareness Training
- Backup Testing

## Lessons Learned

- Action items require owners and due dates.
- Decisions should be separated from discussion points.
